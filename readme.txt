Add custom CSS classes to markup via hooks for CSS usage.

== Description ==

Skins adds a simple admin interface to extend the `body_class`, `post_class`, and `comment_class`. Skins provides only the interface and leaves styling to the designer.

### Usage

- Set classes in Appearance &rarr; Skins
- Use the classes in your CSS

### Github

- [github.com/ryanve/skins](https://github.com/ryanve/skins)
